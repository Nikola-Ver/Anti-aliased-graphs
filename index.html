1. Чтобы открыть файл, нажмите дважды по одному из графиков

2. Чтобы выбрать график для работы с ним, нажмите по нему левой кнопкой мыши

3. Чтобы перемещаться по графику используйте стрелки

4. Чтобы выбрать несколько графиков зажмите кнопку Ctrl и нажмите по нужным вам графикам левой кнопкой мыши

5. Для того, чтобы изменить масштаб, используйте кнопки: a (приблизить), b (отдалить)

6. Чтобы увеличить количество делений сетки нажмите + или -

7. Чтобы выбрать ось, по которой мы будем масштабировать или изменять количество делений сетки, зажмите кнопку Ctrl и нажмите x или y

8. Чтобы убрать сглаживание графика, нажмите правый Shift, чтобы вернуть сглаживание еще раз нажмите правый Shift (предварительно нужно выбрать сам график или графики)

9. Чтобы выбрать вершину, нажмите левый Shift, а затем нажмите на нужную вам вершину левой кнопкой мыши. После выбора вершин, можно выключить данный режим нажав еще раз левый Shift

10. Чтобы изменить скорость перемещения по графику, нажмите комбинацию клавиш Ctrl + Shift + Alt, после чего введите необходимую скорость (стандартная 10). Чтобы закрыть, нажмите ту же комбинацию

11. Если хотите изменить толщину сетки, зажмите комбинацию клавиш Ctrl + I, затем нажмите кнопку a (чтобы увеличить толщину), или кнопку b (чтобы уменьшить)
